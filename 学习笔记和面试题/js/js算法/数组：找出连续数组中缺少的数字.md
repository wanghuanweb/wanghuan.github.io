头目：
给定某无序数组，其包含了 n 个连续数字中的 n – 1 个，已知上下边界，要求以O(n)的复杂度找出缺失的数字。

思路：
1.求出现在数组的和
2.根据上下边界值，求若连续数组的和。利用n(n+1)/2-m(m+1)/2

```
function findMissingNumber(arr,upper,lower){
    var len = arr.length,
        sum = 0,
        sum2 = 0,
        i = 0;

    for(;i < len;i++) {
        sum += arr[i];
    }

    sum2 = upper*(upper+1)/2 - lower*(lower+1)/2;
    return sum2 - sum;
}
```
