使用对象来存储，key-value。
key是字符，value是出现的重复次数。
遍历字符串之后得到一个key-value的对象
遍历对象得到value最大的key值。

```
function mostNum(arr) {
    var obj = {},
        len = arr.length,
        maxValue = arr[0];

    for(var i = 0 ;i < len;i++) {
        if(!obj[arr[i]]){
            obj[arr[i]] = 1;
        } else {
            obj[arr[i]] += 1;
        }
    }

    for(var i in obj) {
        if(obj[i] > maxValue) {
            maxNum = i;
            maxValue = obj[i];
        }
    }
    return maxNum;
}
```
